# Introduction to Programming 💻

Welcome to my **Introduction to Programming** repository! This repo serves as a personal archive for all the C code I am writing and learning throughout my Computer Science Fundamentals course. 

## 📂 Repository Structure

To keep everything clean and accessible, this repository is highly structured. Here is how the files are organized:

* **`Modules/`**: Contains core conceptual code and notes, separated by the specific modules taught in the course (e.g., Module-01, Module-02).
* **`Practice_Days/`**: Dedicated folders for routine practice exercises. These are built to reinforce the concepts learned in the modules.
* **`Problem_Solving/`**: Contains my solutions to algorithmic challenges, logic building tasks, and specific problem-solving assignments.

## 🛠️ Language & Tools
* **Language:** C
* **Concepts Covered:** Basic syntax, control flow, functions, arrays, pointers, memory management, and foundational logic building.

## 🚀 How to Navigate
Feel free to explore the directories. If you are looking for how I tackled a specific logic problem, check the `Problem_Solving` folder. If you want to see the progression of my learning, follow the `Modules` sequentially!
